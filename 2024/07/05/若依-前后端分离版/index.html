什么是若依？

开源项目，学习开源项目的目标：

1.用，减少自己工作量

2.学习优秀的开源项目底层的编程思想，设计思路，提高自己的编程能力

官网：https://www.ruoyi.vip/

基础：Spring Boot + Vue

环境要求：

  1. JDK 1.8+
  2. MySQL
  3. Redis
  4. Maven
  5. Vue
